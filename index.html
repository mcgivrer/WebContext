WebContext by mcgivrer

A php study to provide a MVC+Agile+Conventions oriented framework. Mainly inspired by Ruby on Rails and all these fantastic "less work" frameworks.

WebContext is a demo web site for a "In Construction" framework.

Dependencies

jquery xhtml 1.1 CSS2.1

Install

just download and unzip it on your http server root directory. and enjoy ;)
